About the Project

This project focuses on developing an Azure Migration Cost Calculator with Embedded AI. The tool provides accurate cost predictions and tailored migration recommendations for Azure cloud services, helping stakeholders such as developers, IT teams, and business leaders make informed migration decisions.

Core Capabilities

AI Cost Estimation:

Provides accurate cost predictions and recommendations.

Chatbot Interface:

Allows users to interact naturally and get instant insights.

Detailed Reporting:

Generates reports on migration costs and potential savings.

Azure Pricing APIs:

Ensures current Azure pricing is used for accurate assessments.
